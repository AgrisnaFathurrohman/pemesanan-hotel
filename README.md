
## Hotel Booking Website (Frontend Only)

Hotel Booking Website adalah proyek frontend yang dirancang untuk memberikan gambaran visual mengenai proses pemesanan kamar hotel secara online.
Project ini dibangun menggunakan HTML, CSS, dan JavaScript tanpa backend, sehingga berfungsi sebagai website statis, UI prototype, atau portfolio showcase untuk kebutuhan demonstrasi desain antarmuka dan alur pemesanan.

Project ini dibuat dengan beberapa tujuan:
- Menunjukkan kemampuan dalam frontend development
- Membuat UI/UX website pemesanan hotel yang bersih dan modern
- Menjadi portfolio untuk kebutuhan melamar kerja
- Memberikan contoh website statis tanpa backend untuk demonstrasi
- Melatih penggunaan HTML, CSS, dan JavaScript
## ğŸš€ Fitur Utama
1. Halaman Beranda (Home)
- Tampilan banner hotel (hero section)
- Deskripsi singkat hotel
- Tombol navigasi ke halaman kamar dan pemesanan
- Tampilan responsive untuk mobile dan desktop

2. Daftar Kamar (Rooms Page)
Menampilkan beberapa tipe kamar lengkap dengan:
- Nama kamar
- Foto kamar
- Harga per malam
- Fasilitas kamar (WiFi, AC, breakfast, dll.)

3. Detail Kamar
- Informasi lebih lengkap mengenai fasilitas, kapasitas, dan deskripsi kamar
- Tombol "Pesan Sekarang" untuk menuju form pemesanan

4. Form Pemesanan (Booking Form)
Input data seperti:
- Nama lengkap
- Email
- Check-in dan check-out
- Jumlah tamu
- Pilihan tipe kamar
- Validasi dasar menggunakan JavaScript

Catatan: Data tidak disimpan ke server

5. Desain Responsif
- Layout otomatis menyesuaikan tampilan mobile, tablet, atau desktop

6. Interaksi Menggunakan JavaScript
- Validasi form
- Pop-up alert setelah pemesanan berhasil dikirim (simulasi)
- Navigasi dinamis pada elemen tertentu

7. Tanpa Backend
- Tidak ada database atau server-side processing
- Semua fitur bersifat simulasi untuk keperluan desain
